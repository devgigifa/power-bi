# Power BI Learning Path – Alura

This repository contains all the projects developed during the **Power BI Learning Path** offered by **Alura**.

The learning path covers the complete Power BI workflow, from data import and transformation to data modeling, DAX calculations, and interactive dashboard creation.

## Learning Path Overview

### 1. Getting Started with Power BI
- Introduction to Power BI Desktop
- Creating the first dashboard
- Understanding the Power BI interface
- Building basic visualizations

### 2. Data Preparation with Power Query
- ETL (Extract, Transform, Load) processes
- Data cleaning and transformation
- Working with the M language
- Merging, pivoting, and reshaping data
- Best practices for data preparation

### 3. Data Modeling and DAX
- Creating calculated columns and measures
- Writing DAX formulas
- Working with variables
- Table functions
- Using the `CALCULATE` function
- Building dynamic calculations

### 4. Data Visualization
- Designing effective dashboards
- Choosing appropriate charts
- Data visualization principles
- Interactive reports
- Data analysis using Power BI

## Technologies

- Power BI Desktop
- Power Query
- DAX (Data Analysis Expressions)
- Data Modeling
- ETL
- Data Visualization
